## ソース管理

nrm は npm ソース管理ツールで、npm ソースを簡単に切り替えることができます。

```bash
npm install -g nrm
nrm ls 
npm use xxx
```

npm で設定を確認： `npm config get registry`



## パッケージ管理

pnpm は高速でディスクスペースを節約するパッケージ管理ツールです。

```bash
npm install -g pnpm
```

## node バージョン管理

nvm は node バージョン管理ツールで、node バージョンを簡単に切り替えることができます。

```bash
npm install -g nvm
```
