# 记录一些好用的 npm 包

> 本页面长期更新

## 源管理

nrm 是一个 npm 源管理工具，可以方便地切换 npm 源。

```bash
npm install -g nrm
nrm ls 
npm use xxx
```

用npm查看设置： `npm config get registry`



## 包管理

pnpm 是一个快速、节省磁盘空间的包管理工具。

```bash
npm install -g pnpm
```

## node 版本管理

nvm 是一个 node 版本管理工具，可以方便地切换 node 版本。

```bash
npm install -g nvm
```
