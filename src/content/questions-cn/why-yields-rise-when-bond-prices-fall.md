# 为什么债券价格下跌时，收益率会升高？

债券里最让新手困惑的一件事就是：

- 债券价格跌了，怎么“收益率”反而上去了？

这看起来很矛盾，但本质是一个很简单的比例关系：债券现金流相对固定，而价格会变；价格一变，按价格算出来的回报比例（收益率）就跟着变。

## 先给结论

**债券价格下跌时，收益率会上升，因为收益率是用“固定的票息/现金流”除以“当前市场价格”算出来的。**

价格越低，分母越小，算出来的比例就越大；反之亦然。

## 最直观的公式：当期收益率

用当期收益率（current yield）来理解最简单：

```
当期收益率 = 年度票息 ÷ 当前价格
```

### 一个数字例子

假设一张债券每年付 50 美元利息：

- 价格 1,000 美元：50 ÷ 1,000 = 5%
- 价格跌到 900 美元：50 ÷ 900 ≈ 5.56%
- 价格再跌到 800 美元：50 ÷ 800 = 6.25%

票息没变，但你用更少的钱买到同样的现金流，所以“按投入金额算的回报比例”就更高。

## 那价格为什么会跌？常见触发因素

债券价格下跌通常对应市场要求的回报变高（收益率上行），常见原因包括：

### 1）市场利率上升（最常见）

利率上升时，新发行债券能给更高票息/收益率。旧债券的票息固定，想卖出去只能降价。

### 2）信用风险上升

如果市场担心发行人违约风险变大，会要求更高风险补偿（更高收益率）。收益率要上去，价格就得下来。

### 3）通胀预期上升

通胀会侵蚀未来现金流的购买力。通胀预期上升时，投资者往往要求更高名义收益率来补偿。

### 4）供需变化

即使基本面没变，某些时点的供需（比如资金赎回、再平衡、监管/杠杆约束）也可能导致集中抛售，从而压低价格、抬高收益率。

## 用现值（PV）从底层再确认一次

债券价格本质上是未来现金流的现值：

```
PV = C/(1+r)¹ + C/(1+r)² + ... + C/(1+r)^n + F/(1+r)^n
```

r 可以理解为市场要求的回报/折现率。

- r 上升 → 折现更“狠” → 现值更低 → 价格下跌
- r 下降 → 折现更“松” → 现值更高 → 价格上涨

这也是为什么债券价格与收益率（或利率）会反向。

## 到期收益率（YTM）视角：更接近“真实回报”

当期收益率只是“票息/价格”的粗算。更完整的指标是 **到期收益率（YTM）**，它会把以下因素都算进去：

1. 持有期间的全部票息
2. 折价/溢价买入带来的到期资本利得或损失
3. 现金流时间价值

因此，当债券价格下跌时：

- 当期收益率会上升
- YTM 也通常会上升（因为你用更低的价格买入，同样的现金流给你更高的年化回报）

## 久期：为什么长期债的变化更剧烈

债券价格对利率变化的敏感度常用**久期（duration）**来近似：

- 久期越高（通常期限更长、票息更低）→ 价格对利率更敏感
- 久期越低 → 价格波动更小

所以同样是收益率上行 1%，长期债往往跌得更明显。

## 常见误区

1. **“收益率上升一定是好事”**：收益率上升往往意味着价格在下跌；如果你需要卖出，可能会亏损。
2. **“收益率高就一定更划算”**：收益率高可能是利率高，也可能是信用风险高，要分清来源。
3. **“债券很稳”**：债券现金流更可预测，但价格并不固定，久期越长波动越大。

## 小结

债券价格下跌时收益率上升，本质就是一个比例关系 + 一个现值折现逻辑：

- 票息（现金流）相对固定
- 价格变化会把收益率“倒算”成相反方向
- 市场利率、信用风险、通胀预期、供需冲击都会推动价格变化

理解这条关系，你就能更从容地解读“收益率上行/下行”的新闻，也能更合理地管理债券久期与利率风险。
